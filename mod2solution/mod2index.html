My First Responsive Page

HTML

It stands for Hyper Text Markup Language.
H: A text that contains link to other texts.Sometimes they link back to the original doc.
M: It is all about content(the data will be published using tags)
L: This is implies that there's some syntax(basically there's a write or wrong way to code.)
Basically, HTML provides the structure or what component do the docs have.

CSS

It stands for Cascading Style sheets.This technology provides styling capability to HTML structure in drastically different ways.It works by associating rules to HTML elements.It is basically concerned with colors,layouts, font-size,any stylistics type of thing.

Bootstrap

It is the most popular framework on github for HTML, CSS & JS.It is a 12 column layout and is used for developing responsive mobile first projects on the web.It is too big and has lots of features but we can use selective download.
However,in this first page design we're using media query properties rather than bootstrap!
